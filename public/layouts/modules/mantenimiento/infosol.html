Instrucciones para la Gestión de Problemas y Redirecciones

El formulario de solicitud está diseñado para redirigir problemas a las áreas correspondientes. A continuación, se describen los tipos de problemas y a qué área o departamento se dirigirán:

  • Problemas de Mantenimiento: Si seleccionas el tipo de problema "Mantenimiento", se redirigirá al área de mantenimiento, donde se atenderán solicitudes relacionadas con equipos, instalaciones, sistemas eléctricos, y servicios generales del hospital.
  • Problemas de Arquitectura: Al seleccionar "Arquitectura", el problema será redirigido al departamento de arquitectura, que se encargará de solicitudes relacionadas con diseño de espacios, modificaciones estructurales y planificación de áreas del hospital.
  • Problemas de Informática: Si eliges "Informática", el problema será gestionado por el área de informática, que atenderá solicitudes relacionadas con soporte técnico, infraestructura de TI, sistemas de software, y redes.
  • Problemas de Ingeniería Clínica: Si seleccionas "Ingeniería Clínica", el problema será redirigido al departamento de ingeniería clínica, que manejará solicitudes relacionadas con equipos médicos, calibraciones, mantenimiento preventivo y soporte técnico especializado en tecnología médica.

Nota: Asegúrate de seleccionar el tipo de problema correcto para garantizar que se dirija al departamento apropiado y se resuelva de manera eficiente.
